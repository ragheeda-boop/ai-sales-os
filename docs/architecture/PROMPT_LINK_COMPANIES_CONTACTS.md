# مهمة: ربط جهات الاتصال بالشركات في Notion

## الهدف
ربط كل سجل في قاعدة بيانات "Contacts" بالشركة المناسبة في قاعدة بيانات "Companies" عبر حقل العلاقة (Relation) باستخدام مفتاح المطابقة **Apollo Account Id**.

---

## معلومات قواعد البيانات

### 🏢 Companies (الشركات)
- **Database ID:** `331e04a62da74afe9ab6b0efead39200`
- **Data Source:** `collection://79a7f859-07f8-40a0-bfcf-6b17f2742c8e`
- **عدد السجلات:** 3,606 شركة
- **حقل العنوان:** `Company Name`
- **مفتاح المطابقة:** `Apollo Account Id` (حقل نصي)

### 👤 Contacts (جهات الاتصال)
- **Database ID:** `9ca842d20aa9460bbdd958d0aa940d9c`
- **Data Source:** `collection://ac1dc287-a97d-4dea-ae62-7415e8118514`
- **عدد السجلات:** 5,901 جهة اتصال (5,898 لديهم Apollo Account Id، و3 فارغين)
- **حقل العنوان:** `Full Name`
- **مفتاح المطابقة:** `Apollo Account Id` (نفس القيمة الموجودة في جدول Companies)
- **الحقل المطلوب تحديثه:** `Company` — وهو حقل Relation يشير إلى Companies

---

## منطق المطابقة

كل جهة اتصال (Contact) وكل شركة (Company) لديهم حقل اسمه `Apollo Account Id`.
القيمة هي نفسها في الجدولين — هذا هو مفتاح الربط.

**مثال:**
- الشركة "Almoosa Doors" → Apollo Account Id = `68b41ed42734cd00012fb6c5`
- جهة الاتصال "Yasser Almoosa" → Apollo Account Id = `68b41ed42734cd00012fb6c5`
- **النتيجة:** ربط Yasser بـ Almoosa Doors عبر حقل Company

## خطوات التنفيذ

### الخطوة 1: بناء خريطة الشركات (Company Map)

**الهدف:** استخراج كل الشركات من Notion وبناء قاموس: `{Apollo Account Id → رابط صفحة الشركة}`

**المشكلة:** الاستعلام يرجع 100 سجل فقط كحد أقصى بدون pagination.

**الحل:** استخدام فلاتر على حقل `Apollo Account Id` بأسلوب البادئة (prefix filtering):

1. أنشئ view مساعد على قاعدة Companies:
```
notion-create-view:
  database_id: 331e04a62da74afe9ab6b0efead39200
  data_source_id: collection://79a7f859-07f8-40a0-bfcf-6b17f2742c8e
  name: _CompanyMapper
  type: table
  configure: SHOW "Apollo Account Id"
```

2. لكل بادئة، حدّث الفلتر واستعلم:
```
notion-update-view:
  configure: FILTER "Apollo Account Id" CONTAINS "<prefix>"; SHOW "Apollo Account Id"

notion-query-database-view:
  view_url: https://www.notion.so/331e04a62da74afe9ab6b0efead39200?v=<VIEW_ID>
```

3. من كل نتيجة، استخرج:
   - `Apollo Account Id` → من حقل البيانات
   - `url` → رابط صفحة الشركة في Notion

```
- خزّن: company_map[r['Apollo Account Id']] = r['url']
```

### الخطوة 2: بناء قائمة جهات الاتصال

**الهدف:** استخراج كل جهات الاتصال من Notion مع: رابط الصفحة + Apollo Account Id + حالة الربط الحالية

**نفس الأسلوب:** أنشئ view مساعد على Contacts وفلتر بالبادئة:

```
notion-create-view:
  database_id: 9ca842d20aa9460bbdd958d0aa940d9c
  data_source_id: collection://ac1dc287-a97d-4dea-ae62-7415e8118514
```

### الخطوة 3: تنفيذ الربط

لكل جهة اتصال:
1. خذ `Apollo Account Id` من بيانات جهة الاتصال
2. ابحث عنه في خريطة الشركات (Company Map)
3. إذا وجدته **و** حقل Company فارغ حالياً:

```
notion-update-page:
  page_id: <معرف صفحة جهة الاتصال — بدون شرطات>
  command: update_properties
  properties: {"Company": "[\"https://www.notion.so/<معرف صفحة الشركة>\"]"}
```

4. سجّل النتيجة (تم الربط / تم تخطيه / خطأ)

**نفّذ التحديثات 3-5 بالتوازي** لتسريع العملية.

### الخطوة 4: التحقق والتقرير

بعد الانتهاء:
1. استعلم من view "By Company" في Contacts لتتأكد
2. قدّم تقرير نهائي:
   - عدد السجلات المربوطة
   - عدد السجلات المتخطاة (مربوطة مسبقاً)
   - عدد اليتامى (بدون Apollo Account Id)
   - عدد غير المتطابقين (Account Id بدون شركة مقابلة)
   - عدد الأخطاء

---

## قواعد حرجة — لا تتجاوزها

1. **لا تكتب فوق أي ربط موجود** — فقط املأ الحقول الفارغة
2. **تأكد أن رابط الشركة يأتي من قاعدة Companies فقط** — لا تستخدم رابط صفحة جهة الاتصال كرابط شركة
3. **تحقق دائماً:** رابط الشركة ≠ رابط جهة الاتصال (يجب أن يكونا صفحتين مختلفتين)
4. **Apollo Contact Id ≠ Apollo Account Id** — حقلان مختلفان تماماً!
5. **لا تخترع بيانات** — إذا ما لقيت تطابق، سجّله وتخطاه
6. **سجّل التقدم** بعد كل دفعة: كم تم ربطه، كم بقي
